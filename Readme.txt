TP Noté - Liste de tâches


Utilisation de l'application dans un format portrait seulement. L'application fonctionne bien avec les formats iPhone7, iPhoneX et iPad donnés. 

L'application est une liste de tâches scrollable. Le bouton "+" en bas de l'écran permet d'ajouter une tâche supplémentaire. Une tâche peut être validée en cochant le checkbox à gauche de son nom. Elle peut aussi être supprimée en cliquant sur la croix à droite de son nom. Le nom d'une tâche est éditable en cliquant dessus. 



_____________________________________________________

Minimum requis : 
- Liste de tâches scrollable
- Ajouter une tâche avec un bouton
- Changer le texte d’une tâche
- Supprimer la tâche ❌
- Valider une tâche 

Fonctionnalités supplémentaires : 
- Compteur du nombre de tâche complète et incomplète 
- Quand une tâche est validée, elle devient grisée et barrée
